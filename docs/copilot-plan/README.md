# Copilot Plan

This folder tracks the reporting, seeding, export, and consistency work for the pharmacy app.

## Scope

- Seed realistic demo data for customers, medications, sales, prescriptions, and stock movements.
- Review data flow from models to controllers to Blade views and correct inconsistencies.
- Add reusable chart components for sales and inventory reporting.
- Add export support for CSV data downloads and PNG graph downloads.
- Keep Docker instructions clear for local and production usage.

## Working Notes

- Use small, reversible changes.
- Prefer existing Laravel and Blade patterns before adding new abstractions.
- Update this plan as milestones land.
